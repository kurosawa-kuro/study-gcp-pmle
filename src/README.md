# PMLE 判断基準教科書（本体）

> Google Cloud Professional Machine Learning Engineer (PMLE) 合格直結の**判断辞書**。
> 技術の網羅説明ではなく、**問題文の制約条件から正解サービス・構成・運用判断を選ぶ**ための教材。
> 設計意図は [`../doc/01_仕様と設計.md`](../doc/01_仕様と設計.md)、スコープと章計画は [`../doc/02_移行ロードマップ.md`](../doc/02_移行ロードマップ.md) を参照。

## 各章の構成（8 項目テンプレート）

全章は次の 8 項目で統一されている（定義は [`../doc/01_仕様と設計.md`](../doc/01_仕様と設計.md) §2.1 が正）。

1. この章で切れる選択肢 / 2. 技術の最短定義 / 3. 試験で選ぶ条件 / 4. 試験で選ばない条件 / 5. ひっかけパターン / 6. 判断フローチャート / 7. 代表問題パターン / 8. 最後に覚える一文

代表問題パターンはすべて**本書の自作シナリオ**（実際の試験問題は転載しない）。
